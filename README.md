<h1>My Maps Application</h1>

<h2>Overview</h2>

<p>This project is a dynamic maps application developed for the ACSC-430 Dynamic Languages course. The application utilizes Django as the web framework and integrates various map APIs to provide interactive mapping functionality. Users can save their routes and sign in to access additional features.</p>

<h2>Features</h2>

<ul>
  <li>Integration of Django web framework for backend development.</li>
  <li>Utilization of map APIs for interactive mapping features.</li>
  <li>User-friendly interface for browsing maps and interacting with map data.</li>
  <li>Ability for users to save their routes.</li>
  <li>User authentication system for signing in and accessing additional features.</li>
</ul>
